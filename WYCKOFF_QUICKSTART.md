# 威克夫策略快速开始指南

## 🚀 快速开始

### 1. 启用威克夫策略

威克夫策略已经默认集成到 `work_flow.py` 中，直接运行即可使用：

```bash
python main.py
```

### 2. 自定义策略组合

如果你想只使用某些威克夫策略，可以编辑 `work_flow.py` 文件，在 `strategies` 字典中注释掉不需要的策略：

```python
strategies = {
    # ... 其他策略
    
    # 威克夫策略（根据需要启用/禁用）
    "威克夫-弹簧": wyckoff_spring.check,              # 经典信号，推荐
    "威克夫-缩量不跌": wyckoff_divergence.check,      # 信号较多
    # "威克夫-SC反弹": wyckoff_selling_climax.check,  # 激进策略，慎用
    "威克夫-吸筹完成": wyckoff_accumulation.check,    # 稳健策略，推荐
}
```

### 3. 测试策略

运行测试脚本查看策略效果：

```bash
python test_wyckoff.py
```

### 4. 推荐组合

**稳健型投资者**：
- ✅ 威克夫-缩量不跌
- ✅ 威克夫-吸筹完成
- ❌ 威克夫-SC反弹（关闭）

**进取型投资者**：
- ✅ 威克夫-弹簧
- ✅ 威克夫-吸筹完成
- ⚠️  威克夫-SC反弹（小仓位）

**激进型投资者**：
- ✅ 威克夫-弹簧
- ✅ 威克夫-SC反弹
- ⚠️  建议配合严格止损

## ⚠️ 重要提示

1. **回测验证**：建议先使用历史数据回测，在 `config.yaml` 中设置 `end_date`
2. **小仓位测试**：实盘使用前建议小仓位测试
3. **严格止损**：任何策略都不能保证100%成功，务必设置止损
4. **大盘环境**：熊市中建议减少使用或降低仓位

## 💡 使用技巧

1. **多策略共振**：同一只股票被多个威克夫策略选中，信号更强
2. **配合其他策略**：威克夫策略可以与原有的"放量上涨"、"均线多头"等策略组合使用
3. **关注日志**：运行时会输出详细的策略匹配信息，包括关键指标
4. **定期复盘**：记录每次信号的后续表现，优化策略参数

## 🆘 常见问题

**Q: 为什么没有找到符合条件的股票？**  
A: 威克夫策略比较严格，特别是弹簧和吸筹完成策略。可以：
- 调整策略参数（在各策略文件中）
- 尝试在不同市场环境下运行
- 扩大样本周期（修改 `data_fetcher.py` 中的 `start_date`）

**Q: 策略运行很慢怎么办？**  
A: 威克夫策略需要较多计算，这是正常的。可以：
- 关闭不需要的策略
- 提高流动性筛选标准（减少待分析股票）

**Q: 如何调整策略参数？**  
A: 打开相应的策略文件（如 `wyckoff_spring.py`），在 `check()` 函数中有详细的参数说明和注释。
